# Support

## Getting Help

Use the following path:

1. Read [README.md](README.md)
2. Read [docs/development/setup.md](docs/development/setup.md)
3. Read [docs/tutorials/first-contribution.md](docs/tutorials/first-contribution.md)
4. Search existing issues before opening a new one

## Best Places for Questions

- GitHub Issues for concrete bugs or documentation problems
- GitHub Discussions for broader design, learning, and architecture questions

## What to Include

- your goal
- your environment
- the exact command you ran
- the exact error you saw
- the file or subsystem involved

## What Not to Post

- secrets
- tokens
- private machine credentials
- vague requests with no reproduction details
